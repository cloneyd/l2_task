Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет числа от 0 до 10 не включительно.
Далее планировщик go вызовет panic и выведет сообщение о возникновении deadlock.

Deadlock - ситуация, в которой группа горутин ожидает друг друга, но ни одна из них не может продолжить свое выполнение.
В данном случае ch - небуферизированный канал, поэтому горутина, которая кладет в этот канал значение, 
блокируется до тех пор, пока это значение не будет считано в другой горутине и наоборот.
Поэтому в програме выше после выполнения цикла планировщик горутин вновь запустит main горутину, которая считает последнее значение из канала,
и далее планировщик запустит горутину с циклом вновь, но та не сможет положить значение в канал, так как цикл уже закончен
а горутина main будет ожидать новое значение в канале ch, в результате возникнет deadlock.  
```
