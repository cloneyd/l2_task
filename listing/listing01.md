Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Вывод программы: [77, 78, 79]

a - массив размерностью 5
b - слайс, в котором лежит указатель на 1 элемент массива a,
а также размер массива - 3,вместимости слайса и массива одинаковы.

Синтаксис создания слайса: array[start:end]
Созданный слайс будет включать в себя элементы с индексами в диапазоне [start; end) 
```
